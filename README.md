# build-intermediate-web-dev
Repository created for Build Spring 2026 to create a chrome extension project.

This chrome extension project will serve as a permanent inspect element on a webpage to allow a user to do several actions that will be useful. Essentially, it is a tool for a 
user to have for several purposes that may include note-taking, organization of a webpage, etc. 

Features will include:

- Changing a webpages elements to adjust for a user's needs.
- Post-it notes that act as tools for the user.
- Drawing elements within a web page
- User adjusted or AI-assisted adjustment of a webpage's paragraph elements
- Removal of elements on a webpage
